Sobre mim

Olá, meu nome é Bruno, sou graduado em Matemática Industrial e mestre em Modelagem e Otimização pela UFG (Universidade Federal de Goiás). Ao longo do tempo, tive diversos contatos com a área da programação, indo de Controladores Lógicos Programáveis (CLP) à kits de robótica da LEGO.

Por ser apaixonado por tecnologia e possuir experiência com programação, passei a ter curiosidade com a área de desenvolvimento front-end. Depois de um tempo, passei comecei a estudar as linguagens de modo autodidata. Atualmente, têm aproximadamente um ano que venho estudando algumas linguagens e ferramentas, para ter um melhor entendimento de suas funcionalidades e principais aplicações. Entre as tecnologias aprendidas, posso citar: JavaScript (com React, Angular, React Native e Node), Java (com Spring, Gradle e Maven), Kotlin, Python e PHP. Além de bancos de dados e gerenciadores de pacotes.

Atualmente estou buscando uma oportunidade de atuar como programador Jr. para poder iniciar minha carreira profissional nessa área. Posteriormente, pretendo criar um projeto para auxiliar quem deseja entrar na área, mas ainda é necessário estudar qual forma pode ser mais viável/eficiente. Caso tenha se interessado em algo, tenha qualquer sugestão, ou queira me conhecer melhor, entre em contato pelo LinkedIn ou e-mail.

Portifólfio

A seguir, estão alguns projetos criados para praticar os conceitos aprendidos, ou que foram realizados em algum curso/evento.


Flappy Bird

Réplica desse jogo que foi uma "febre coletiva" utilizando Canvas e JavaScript. O jogo está totalmente funcoional e pode ter algumas configurações modificadas.

Ecoleta

O Ecoleta é uma aplicação voltada à coleta de materiais recicláveis e lixo orgânico. Foi construido com React.js e Node.js. A aplicação possui as versões front-end.

Delivery

O Delivery é uma aplicação no sistema delivery. Como um ponto diferencial ela possui um mapa para que o cliente possa selecionar para onde enviar o pedido. A plataforma foi desenvolvida com Java e React.

Podcastr

O Podcastr é uma aplicação com o foco de ececutar Podcast. Ele está tulizando um banco de dados locais, mas posteriormente poderá acessar os dados de uma api. A api não foi desenvolvida ainda.
