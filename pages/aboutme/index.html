2014 - 2016

Mestrado

Participei de um grupo que estudava tecnologias assistivas para membros superiores. O objetivo era procurar soluções acessíveis para alcançar a maior quantidade de pessoas. Eram utilizadas impressão 3D e Arduino para reduzir o valor das próteses. Um problema que era muito comum, era a dimensão dos modelos serem muito desproporcionais. Devido a tal, fiquei responsável por estudar a relação entre o tamanho dos membros e a altura das pessoas.

Para facilitar o entendimento desta relação, foi desenvolvido um modelo visual. Baseado na literatura, foi possível estimar uma relação e a partir dela, foi desenvolvido um modelo visual. Devido a vários fatores, esse modelo não é o ideal, mas atende a preocupação inicial de melhorar o dimensionamento das próteses. Este modelo pode ser acessado clicando aqui.

Cursos

Graduação

Além desses cursos, também me graduei em Matemática industrial pela UFG - Regional Catalão (Atualmente - UFCat). Onde comecei a ter um convívio com a área de programação a partir das disciplinas de Algoritmo e Programação de Computadores (APC), e Programação Orientada a Objeto (POO), entre outras. No decorrer do curso, várias disciplinas necessitavam de programação para solucionar alguns problemas (Modelagem Matemática, Calculo Numérico, etc.).

Outro fator que contribuiu para o reforço do aprendizado de programação durante a graduação, foi participar de um Projeto de Robótica Educacional. Nele foram ensinadas algumas técnicas de montagem e programação com o kit - ®Lego NXT Education. Participei como assistente, portanto auxiliava os alunos durante a execução das atividades.

Além desse projeto, também fui estagiário do Laboratório de Controle e Automação do Departamento de Matemática. Pois já tinha aprendido sobre automação em um curso de eletricista. Eram utilizadas bancadas de automação com Controladores Lógicos Programáveis (CLP), bancadas de sensores, eletropneumáticas, entre outras. Como a automação também utiliza programação, foi mais uma experiência muito importante.

Por ter empregado bastante tempo no projeto de Robótica Educacional e no Laboratório de Automação, aproveitei o conhecimento adquirido para o meu Talho Final de Curso (TFC ou TCC ou PFC). Desenvolvi uma maquete que simulava uma parte de um processo automatizado de montagem veicular. O trabalho foi planejar, criar, e programar toda a planta. O resultado desse trabalho pode ser conferido neste vídeo.
