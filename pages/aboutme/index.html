Estudo de Programação

Em 2020, após um período que havia tirado para analisar em qual área continuaria atuando, comecei ter interesse na área de programação. Por passado por muitas situações que envolviam a programação, percebi que já tinha uma base de lógica e algoritmos. Devido a tal, comecei ver como é o universo da programação e percebi a infinidade de linguagens e ferramentas existentes. Para ver em qual área eu gostaria de atuar, comecei consumir conteúdo de diferentes setores.

Nesse processo tive contato com diversas linguagens de programação, como: JavaScript, Java, Kotlin, Python, PHP. Também utilizei várias ferramentas como: GitHub, Yarn, npm, React, Angular, TypeScript, Maven, Gradle, entre outros.

Após conhecer um pouco sobre as áreas de desenvolvimento web, mobile e data science, escolhi atuar com o desenvolvimento web e mobile. 2014 - 2016

Mestrado

Participei de um grupo que estudava tecnologias assistivas para membros superiores. O objetivo era procurar soluções acessíveis para alcançar a maior quantidade de pessoas. Eram utilizadas impressão 3D e Arduino para reduzir o valor das próteses. Um problema que era muito comum, era a dimensão dos modelos serem muito desproporcionais. Devido a tal, fiquei responsável por estudar a relação entre o tamanho dos membros e a altura das pessoas.

Para facilitar o entendimento desta relação, foi desenvolvido um modelo visual. Baseado na literatura, foi possível estimar uma relação e a partir dela, foi desenvolvido um modelo visual. Devido a vários fatores, esse modelo não é o ideal, mas atende a preocupação inicial de melhorar o dimensionamento das próteses. Este modelo pode ser acessado clicando aqui.
