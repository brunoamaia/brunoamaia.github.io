Gerenciador de Produção

Projeto pessoal com a finalidade de gerenciar/acompanhar um sistema produtivo. O projeto ainda está em desenvolvimento e como ele está sendo produzido do zero, seu desenvolvimento será lento. Devido ao fato de criar/ajustar todo o conteúdo, além da adição de funcionalidades.

Para o desenvolvimento de toda a estrutura do Frontend do Gerenciador, foi utilizado apenas as ferramentas básicas do desenvolvimento web (HTML5, CSS3 e JavaScript).

O Backend ainda não foi iniciado, portanto, ainda não foi selecionado nehum sistema de Banco de Dados ainda.

A idéia do projeto, é que o sistema se comunique com algum equipamento ou plataforma, para testar sua robustez.

Posteriormente, será criada uma versão mobile. Mas apenas após a versão web estiver com todas as funcionalidades implememtadas.

A versão web pode ser "atualizada", ou receber uma nova versão utilizando algum framework (React ou View). Esta ação será para treinar o uso do framework
