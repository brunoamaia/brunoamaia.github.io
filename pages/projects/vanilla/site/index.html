Versões do site

Serão apresentadas algumas das etapas de melhoria (que refletem o aprendizado adquirido).

Versão 0

Criação das primeiras páginas estáticas com a utilização de HTML e CSS. A navegação entre as páginas ocorria pelos botões posicionados no final das páginas.

Versão 1

Criação do menu de navegação (navbar) e ajuste do conteudo para ser responsivo. Foi criado um menu de navegação para dispositivos mobile e utilizado jQuery para realizar a transição entre os estados de ativo e inativo.

Versão 2

Alteração no esquema de cores do site. Utilização de arquivos genéricos para o estilo e funcionalidades do site. Remoção do jQuery para utilizar os recursos do JavaScript ES6. E adição de meta tags para o SEO do site e adição de recursos de monitoramento da Google.

Versão 3 (Atual)

Criação de animação no menu do navbar para mobile. Utilização semantico do HTML, para auxiliar ainda mais no SEO. Mudança no esquema de cores para melhorar a leitura e criação de "módulos" para aceler na implememtação das páginas.
