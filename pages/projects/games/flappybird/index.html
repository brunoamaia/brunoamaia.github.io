Game - Flappy Bird

Para jogar, clique no jogo.

O jogo está apresentando diferença de performance em diferentes dispositivos. Então seu jogo pode estar mais lento, ou mais rápido que o "normal". Por isso foram adicionadas as dificuldades. Para tornar possível jogar em todas as condições.
Obs: em dispositivos mobile, as notificações podem atrapalhar.

Jogo criado utilizando Canvas API que é um elemento do HTML5 que trabalha com manipulação de gráficos. Para criar os gráficos e permitir a interação com o jogo, está sendo utilizado o JavaScript.

Informações adicionais

Este jogo foi criado ao acompanhar a série: Criando um Flappy Bird do ZERO!!! do Dev Soutinho. A programação não segue completamente a série, pois preferi seguir um caminho diferente em algumas partes (E também porque me empolquei e acabei "finalizando" o jogo antes da série terminar kkkkkkkk).

Pelo fato do jogo estar em uma página estática, existem muitas limitações de segurança. Portanto, saiba que existem formas mais fáceis de conseguir pontuações altas kkkkkkk. Mas o desafio do jogo é testar a sua coordenação motora, lembre-se disso.

Ainda existem alguns "Bugs" que não foram corrigidos. Mas praticmente não atrapalham a jogabilidade no computador (que foi a plataforma que foi utilizada para testar no desennvolvido. O funcionamento mobile necessita de otimizações).
