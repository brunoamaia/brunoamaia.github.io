Delivery

Projeto desenvolvido durante a segunda semana Dev Superior. Que foi promovida entre os dias 04 e 08 de janeiro de 2020.

O Delivery é uma aplicação com o intuito criar toda a estrutuda necessária para uma aplicação nos dias atuais. É composto pelo backend, frontend e um aplicativo mobile.

O servidor foi desenvolvido com Java (utilizando Spring boot) e está hospoedado na Heroku. E por utilizar a versão gratuita, pode levar alguns minutos para sair do modo de suspensão.

O frontend (cliente, ou páginas do navegador) foi criado com React e hospedado na Netlify. A página está sempre disponível, mas os dados requisitados pela API podem demorar um pouco, devido o que informado anteriormente (no servidor).

Já o aplicativo mobile por ser uma versão experimental, não foi hospedade em nenhum local. Posteriormente, será criado um vídeo para mostrar seu funcionamento.

A imagem a seguir, é do frontend e pode ser acessada clicando aqui ou na imagem a seguir. É importante ressaltar, que o conteúdo ainda será modificado.

O projeto foi postado nas plataformas da Vercel e Heroku. Para acessá-lo e ver essas funcionalidades, clique aqui.
