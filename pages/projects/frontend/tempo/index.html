Previsão do tempo

Projeto desenvolvido em um curso da Digital Innovation One (DIO).

O projeto consiste em criar uma interface para uma aplicação que busca os dados sobre tempo na API da Weather (openweathermap). Que é uma API que fornece informações relativas ao clima em praticamente qualquer cidade. Como é utilizada a versão gratuita da API, apenas alguns dados são recebidos.

A aplicação permite realizar a busca da temperatura em ºC, ºF ou ºK, Sendo ºC a busca padrão. É possível procurar a cidade digitando o nome e depois clicando em pesquisar, ou utilizando o autocompletar. Para facilitar seu uso, é possível adicionar algumas cidades na parte de favoritos.
