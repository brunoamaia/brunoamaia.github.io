Gamequiz

Projeto desenvolvido durante a Imersão React Next.JS. Que foi promovida pela Alura entre os dias 25 e 29 de janeiro de 2021.

A proposta da semana de imersão era criar um quiz utilizando React e Next. O layout foi estruturado de forma que os parametros (cores, imagens e alguns textos) fossem todos adquiridos a partir de um arquivo JSON.

Esta estruturação se deu, devido o fato da aplicação utilizar "lambda server". Essa funcionalidade permite criar uma API que diponibiliza os dados a partir de um arquivo JSON. Desse modo, podemos acessar os dados de outros projetos e se utilizarem o mesmo padrão de formatação, conseguimos executar esses quizes dentro da nossa estrutura.

O quiz pode ser acessado clicando aqui. Foi desenvolvido um quiz baseado em jogos clássicos de arcade, como pode ser observado na imagem a seguir.
