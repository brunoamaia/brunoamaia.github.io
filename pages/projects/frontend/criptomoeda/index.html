Criptomoeda

Projeto desenvolvido em um curso da Digital Inovation One (DIO).

O projeto consiste em criar uma interface para uma aplicação que busca os dados sobre Criptomoedas na API da Coin Market Cap. Que é uma API que fornece inúmeras informações relativas a várias Criptomoedas. Como é utilizada a versão gratuita da API, apenas alguns dados são recebidos.

No primeiro acesso, os dados são requisitados da API. Após receber os dados, estes são salvos na seção do navegador. Deste modo, os próximos carregamentos da página não consultam a API. Esta funcionalidade foi implementada para diminuir o tráfego de dados na rede e não "estourar" o limite de acesso a API (por utilizar a versão grátis).

A aplicação permite ordenar os dados de acordo com as informações fornecidas pela API. Para poder focar nas funcionalidades, foi utilizado Bootstrap, o que torna a criação do layout mais rápida. Foi implementada a opção de adiconar ou remover algumas colunas para melhorar a visualização dos dados. Para praticar alguns conhecimentos com JavaScript, foram adicionadas opções que alteram o layout e também a necessidade de acessar a API.

A aplicação pode ser acessada clicando aqui. Mas para seu funcionamento, é necessário seguir algumas regras que estão informadas na página. Estas regras são essenciais para que a API possa funcionar.

Caso queira olhar o código e quem sabe contribuir, segue o Link para o Repositório no GitHub.
